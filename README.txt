Aplikacja umożliwiająca wypożyczanie samochodów.

Technologie:
JAVA
SQL przy wykożystaniu biblioteki JDBC

Aplikacja zawiera:
Logowanie i Rejestrowanie użytkowników.
Podczas rejestrowania użytkownicy dodawani są do Bazy Danych.
Dodawnie samochodów do wypożyczalni
Zalogowany użytkownik może wyświetlić listę dostępnych aut.
Po wybraniu auta na wynajem, samochód znika z listy dostępnych aut, ale pozostaje w bazie danych.
